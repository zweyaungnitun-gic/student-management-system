GIC Career Gateway မိတ်ဆက်

GIC Career Gateway သည် ၂၀၁၁ ခုနှစ်တွင် စတင်တည်ထောင်ခဲ့ပြီး၊ မြန်မာနိုင်ငံအတွက် အကောင်းဆုံး IT Solution များကို ပံ့ပိုးပေးနိုင်ရန် IT ပညာရှင်များအား မွေးထုတ်ပေးနိုင်ရန် ရည်ရွယ်သည်။ ၎င်းသည် GIC Group မှ တည်ထောင်ထားသော သင်တန်းကျောင်းတစ်ခုဖြစ်ပြီး၊ သင်တန်းကျောင်းသား/သူများအား ၂၀၁၃ ခုနှစ်မှစတင်၍ သင်ကြားပေးခဲ့ပါသည်။ GIC Group သည် ၂၀၁၁ ခုနှစ်မှစ၍ ယနေ့အထိ နိုင်ငံတကာအဆင့်မီ Training Program များကိုပြုလုပ်လျက်ရှိပြီး၊ ကျောင်းသား/သူများ၏ အောင်မြင်မှုကို အမြဲဦးစားပေးသည်ဟု ဆိုပါသည်။ IT ကျွမ်းကျင်မှုမရှိသေးသော သင်တန်းကျောင်းသား/သူများအတွက် Career Path အတွက် အထောက်အကူဖြစ်စေပါသည်။

IT ပညာရပ်များတွင်ကျွမ်းကျင်မှုရှိပြီး စိတ်ဝင်စားသူများ၊ မိမိတို့၏ပညာရေးနှင့်အလုပ်အကိုင်လမ်းကြောင်းများအတွက် ရည်ရွယ်ချက်ရှိသူများ၊ IT လုပ်ငန်းခွင်တွင် အသက်မွေးဝမ်းကျောင်းပြုလုပ်လိုသူများအတွက် Scholarship Program များလည်း ပံ့ပိုးပေးလျက်ရှိပါသည်။

ဘယ်သူတွေအတွက်လဲ？ဘယ်သူတွေတက်‌သင့်လဲ?

အနာဂတ်တွင် ဂျပန်နိုင်ငံသို့ ITပညာရှင်အနေဖြင့် သွားရောက်လိုသူလူငယ်တိုင်းအတွက်

  • ၁) ဘွဲ့ရပြီးခါစ၊ သို့မဟုတ် ဘွဲ့ရပြီး (၃)နှစ်အတွင်းရှိသူများ
  • ၂) IT ပညာရပ်ကို စိတ်ဝင်စားပြီး ဘွဲ့မရသေးသောသူများ
  • ၃) IT & Japanese Language ကို zero level မှ စတင်သင်ယူလိုသူများ
  • ၄) IT background ရှိပြီး၊ IT နယ်ပယ်သို့ ပြောင်းလဲလိုသူများ၊ IT လုပ်ငန်းတွင် အတွေ့အကြုံရှိ၍ Japan Experience ရယူလိုသော ဂျပန်ကို စိတ်ဝင်စားသူများ
  • ၅) ဂျပန်နိုင်ငံ Language School များမှဘွဲ့ရပြီးသော သင်တန်းသား/သူများ
  • ၆) ဂျပန်နိုင်ငံတွင် IT Engineer အဖြစ် အလုပ်လုပ်ကိုင်လိုသူများ

GIC Career Gateway ကိုဘာကြောင့်ရွေးချယ်သင့်တာလဲ?

  • ဂျပန်နိုင်ငံ၊ တိုကျိုမြို့ အခြေစိုက် IT ကဏ္ဍမှ ဦးဆောင်ဖွင့်လှစ်ထားသည့် Liberal Arts, IT & Japanese Training Center ဖြစ်ခြင်း
  • သင်တန်းပြီးဆုံးနောက် GIC ကွန်ရက်အဖွဲ့အစည်းများကမှီ အတွင်းတွင် အလုပ်လုပ်ကိုင်ခွင့် ရရှိအောင် အခွင့်အလမ်းများဖန်တီးပေးခြင်း
  • OJT Project များလုပ်ဆောင်ရန်အတွက် GIC ကွန်ရက်တွင် အလုပ်လုပ်ကိုင်လိုသူများအား GIC ဝန်ထမ်းများမှ support ပေးနိုင်ခြင်း
  • အခြား ကျောင်းများနှင့်မတူဘဲ အခြေခံမှစ၍ လက်တွေ့လုပ်ဆောင်နိုင်သည့်အထိ သင်ကြားပေးခြင်း
  • တက်ရောက်လုပ်ငန်းခွင် အတွေ့အကြုံများကို ရရှိစေခြင်း
  • IT နယ်ပယ်သားသမီးများအတွက်သာမက IT Professional တစ်ယောက်အတွက် ရှိသင့်သည့် skill များ၊ အတွေ့အကြုံများနှင့် ဉာဏ်ရည်ဉာဏ်သွေး မြင့်မားစေခြင်း

All in One 2 Years Course

Career Path with Career Gateway

GIC Career Gateway မှသင်တန်းပြီးသည့်နောက် GIC ကုမ္ပဏီတွင် Internship အနေဖြင့် ဝင်ရောက်လုပ်ကိုင်နိုင်မည် ဖြစ်ပါသည်။ Internship ကာလအတွင်းတွင်လည်း ကုမ္ပဏီမှ လစာ ၁၀၀,၀၀၀ ကျပ်အပြင် လမ်းစရိတ်ကိုပါ ထောက်ပံ့ပေးမည်ဖြစ်ပါသည်။ Internship ကာလမှာ ၆ လ ကြာမြင့်မည်ဖြစ်ပါသည်။ Internship ကာလပြီးဆုံးသည့်နောက် ကုမ္ပဏီဘက်မှ အကဲဖြတ်၍ ပြဿနာတစ်စုံတစ်ရာမရှိပါက အလုပ်ခန့်အပ်မည် ဖြစ်ပါသည်။ ထို့အပြင် မိမိ skill နှင့် ကိုက်ညီမှုရှိသော အခြားမိတ်ဖက်ဂျပန် IT ကုမ္ပဏီများနှင့်လည်း မိတ်ဆက်ပေးမည်ဖြစ်သောကြောင့်အလုပ်အကိုင်ရရှိနိုင်ပါသည်။
